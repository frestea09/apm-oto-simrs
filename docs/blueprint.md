# **App Name**: BPJS Verification Simplified

## Core Features:

- Frista/After.exe Launcher: One-click launch buttons for Frista and After.exe applications.
- Automated Login: Automatically logs into Frista and After.exe using predefined credentials (Username: `1002r006th`, Password: `#Bandung28`).
- BPJS Number Input: Easy-to-use form for inputting BPJS and booking numbers with keyboard or barcode scanner support.
- Automated Fingerprint Verification: Automatically triggers fingerprint verification in the desktop applications upon BPJS number input.
- Automated Ticket Printing: Automatically prints the patient ticket upon successful verification.
- Accessibility Features: Includes high contrast mode, large buttons, clear text, and voice prompts for disabled users.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to convey trust and stability, referencing healthcare environments.
- Background color: Light Gray (#ECEFF1) for a clean, accessible, and non-distracting interface.
- Accent color: Orange (#FF9800) to highlight key CTAs and interactive elements, contrasting against the primary color.
- Body and headline font: 'PT Sans', a sans-serif, providing a modern and accessible typeface that will work well for a diverse set of users including the elderly and the disabled. 
- Use clear, recognizable icons to represent actions and statuses within the application.
- Prioritize a simple, intuitive layout with large, easily tappable buttons to accommodate users with motor impairments.
- Subtle animations to guide the user through each step of the verification process.
